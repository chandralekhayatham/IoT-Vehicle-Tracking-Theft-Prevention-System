# IoT Vehicle Tracking & Theft Prevention System

## Overview

The IoT Vehicle Tracking & Theft Prevention System is a Python-based IoT simulation that monitors vehicle location, checks geofence boundaries, and generates alerts when unauthorized movement is detected.

---

## Features

- GPS Coordinate Simulation
- Google Maps Link Generation
- Geofence Detection
- Theft Alert Generation
- CSV Data Logging
- Python-Based IoT Simulation

---

## Technologies Used

- Python
- IoT Concepts
- GPS Simulation
- CSV File Handling

---

## How to Run

```bash
pip install pandas matplotlib
python main.py
```
